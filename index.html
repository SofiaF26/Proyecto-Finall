Preguntas introductorias


II.3.1. ¿Qué es un framework?

Un Framework es una estructura base utilizada como punto de partida para elaborar un proyecto con objetivos específicos.


II.3.2. ¿Qué es Bootstrap5 y cuál es su origen?

Bootstrap es un framework de código abierto disponible para HTML, CSS y JS. Bootstrap fue desarrollado por Mark Otto y Jacob Thornton.


II.3.3.¿Qué es diseño responsive en páginas web?

Redimensión de los elementos en una página web para que sean colocados de manera que se adapten al tamaño de cada dispositivo, permitiendo así una visualización adecuada.


II.3.4. ¿Qué son las Github pages?

Plataforma que permite subir repositorios de código para almacenarlo en el sistema de control de versiones Git.


II.3.5. ¿Cuál es la diferencia entre elemento y componente web?

Los componentes web son un paquete de diferentes tecnologías que te permiten crear elementos personalizados reutilizables y los elementos


II.3.6 ¿Cuál es la importancia de documentar o comentar código?

Para guiarnos al momento de documentar nuevo código y tener noción de donde y qué estamos trabajando


II.3.7 ¿Por qué existe la sección de atribución en un sitio web?

Para dar los respectivos créditos y reconocimientos a los creadores.
